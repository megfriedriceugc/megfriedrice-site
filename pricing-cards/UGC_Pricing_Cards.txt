Meg Fried Rice - UGC Pricing Cards (copy for brands)

Last updated: 2026-04-02

How to use this
- Send this doc to brands after they ask for pricing.
- Treat base packages as the starting point.
- Add-ons are requested separately and quoted based on campaign details.

What I deliver (for all packages)
- Fully edited short-form vertical videos for UGC-style posting.
- Story-led creative direction tailored to your brand.
- Included revisions (per package).
- Delivery format and video specs are agreed during the brief.

Turnaround
- Typical turnaround is about a week after filming.
- Timing depends on how many videos are in the package and review cycles.

Rights & usage (the important part)
- All base packages include full organic usage rights for 3 months.
- Paid social and advertising usage, raw files, and extended or exclusive organic usage are available on request and quoted separately.

Paid ads usage vs organic usage
- Organic usage means posting on the brand's own channels (without ad-account licensing for paid campaigns).
- Paid ads usage means you can run the delivered videos in paid campaigns for the agreed time period.
- Whitelisting (when required) is the process of giving ad-account permission so the brand can run the content as you (or with your creator identity) in their ad system.

Whitelisting add-on (if needed)
- Some brands want the ability to run the content inside their ad account.
- If your campaign needs whitelisting, this is typically quoted as a one-time setup (plus the paid usage period).

Raw files / project files
- If requested, I can provide original raw media or project files (if available for the package).
- This is quoted separately because it can affect delivery time and file handling.

Extended organic usage
- If you want more than the included 3 months, extended organic can be added for additional periods (quoted separately).

Exclusive organic rights (optional)
- If you want exclusivity (for a defined competitor/category), this can be added.
- Exclusivity is always quoted separately because it restricts future work.

Suggested add-ons (quoted based on request)
Notes
- When brands request paid usage, I quote usage length and channels.
- If the brand requests both paid usage and whitelisting, I price both as separate line items.

1) Paid ads / advertising usage (common add-ons)
- Paid usage is quoted per video and per usage period.
- Example starting points (you can adjust based on scope):
  - +GBP 175 per video for 30 days paid usage on 1 main platform/channel
  - +GBP 300 per video for 30 days paid usage across multiple channels (e.g. Meta + TikTok)

2) Whitelisting setup (if required)
- +GBP 150 one-time per campaign for whitelisting setup (typical starting point)

3) Raw files / project files
- +GBP 100 per video (typical starting point)

4) Extended organic usage
- Add extra organic duration beyond the included 3 months
- Typical starting point: +GBP 75 per video for an additional 3 months

5) Exclusive organic rights (optional)
- Exclusivity is quoted separately
- Typical starting point: +GBP 250 per video for 3 months exclusivity in a defined category

What to ask the brand (so you quote confidently)
- How many videos are they requesting?
- Do they want organic only, or paid usage too?
- If paid usage:
  - For how long (e.g. 30/60/90 days)?
  - Which channels (Meta, TikTok, etc.)?
  - Do they need whitelisting?
- Do they want raw files/project files?
- Do they need extended or exclusive organic rights?
